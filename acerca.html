Acerca de Este Proyecto

Informacion sobre la tecnologia utilizada y el proposito del proyecto.

Que es este proyecto?

Esta es mi primera pagina web en produccion. Es un proyecto de aprendizaje para entender como funcionan las tecnologias web modernas: frontend, backend, base de datos y deployment.

La pagina permite gestionar una lista de clientes, con datos guardados en una base de datos real (Supabase/PostgreSQL).

Stack Tecnologico

Frontend: HTML, CSS, JavaScript
Base de Datos: Supabase (PostgreSQL)
Hosting Frontend: Vercel
Hosting Backend: Railway
Control de Versiones: Git + GitHub
Asistente IA: Claude Code

Como Funciona el Deploy?

1 Edito el codigo en VS Code
↓
2 git add + git commit
↓
3 git push (sube a GitHub)
↓
4 Vercel detecta el push y despliega automaticamente
↓
5 En 30 segundos esta en produccion!
